Generic univariate summary statistic objects.

UnivariateStatistic API Usage Examples:

UnivariateStatistic:

/* evaluation approach */
double[] values = new double[] { 1, 2, 3, 4, 5 };
UnivariateStatistic stat = new Mean();
System.out.println("mean = " + stat.evaluate(values));
